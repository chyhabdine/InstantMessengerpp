# Roadmap

- **M1: Foundations** – Complete domain modeling, EF Core configuration, JWT auth, and baseline CRUD endpoints (done).
- **M2: Messaging UX** – Enhance message features (edit/delete, typing indicators, delivery/read receipts), add pagination and search.
- **M3: File Handling** – Replace URL-based attachments with storage integration (Azure Blob/S3), virus scanning, and size quotas.
- **M4: Notifications** – Add push/websocket notifications, background workers for offline delivery, and digest emails.
- **M5: Security & Ops** – Rate limiting, audit logging, advanced role policies, secrets management, CI/CD pipeline, staging/production configs.
- **M6: Clients** – Build web/mobile clients consuming the API with real-time updates and offline cache.
