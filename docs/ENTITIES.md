# Entities (10+)

1) **User** ??? core identity used by authentication and profiles.
2) **Role** ??? separates Admin/User permissions.
3) **UserProfile** ??? stores display name, status, presence, avatar.
4) **Session** ??? stores login sessions and access tokens.
5) **Device** ??? tracks the device used for login.
6) **Conversation** ??? private or group chat container.
7) **ChatMember** ??? membership and role inside a conversation.
8) **Message** ??? message payload in a conversation.
9) **Attachment** ??? metadata for message attachments.
10) **Reaction** ??? per-user reaction on a message.
11) **FriendRequest** ??? request/accept flow between users.
12) **Notification** ??? read/unread events for messages and requests.
