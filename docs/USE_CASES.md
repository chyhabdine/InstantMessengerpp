# Use Cases (Messenger)

- **Authenticate** ??? a visitor creates an account and logs in to obtain an access token.
- **Update profile** ??? a user updates display name, avatar, status, and presence.
- **Search directory** ??? a user searches for other users to start a conversation.
- **Create a conversation** ??? a user creates a private or group conversation and adds members.
- **List conversations** ??? a user retrieves the list of their chats with members.
- **Exchange messages** ??? a user sends messages and reads chat history.
- **React to a message** ??? a user adds or updates a reaction on a message.
- **Manage invitations** ??? a user sends/receives friend requests and views contacts.
- **Receive notifications** ??? a user checks notifications for messages and friend requests.
- **Manage sessions** ??? tokens are refreshed and sessions are tracked per device.
