# Team Roles

- **Solution Architect** – Owns overall architecture, quality gates, and security posture.
- **Backend Lead** – Maintains API contracts, database schema, migrations, and service reliability.
- **QA Engineer** – Designs test plans, automates regression checks, validates migrations and seed data.
- **DevOps Engineer** – Owns deployment pipelines, secrets management, observability, and runtime environments.
- **Product Owner** – Curates roadmap, defines acceptance criteria, and prioritizes messaging/notifications features.
